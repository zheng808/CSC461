Microsoft Hololens

Hololens is a pari of mixed reality smartglasses that is developed by Microsoft and first released in 2016.

Figure 1

Mixed Reality

Hololens has been categorized as an mixed reality (MR) device instead of augmented reality (AR) a device. MR is a very similar idea as AR, except that it's more advanced to support virtual object interacting with real world objects. For example, if a virtual TV screen is set to attach to a wall, the virtual TV screen will stay in the same place with respect to the user's view, see Figure 2. This way, Hololen will create an illusion to make user believe the virtual object exist in the real world.

Figure 2


Hardware support

Hololens has installed many cameras and sensors to capture information about the surrounding environment. Four environment understanding cameras are used to quickly scan the environment in front of the user within a range of 120x120 degrees, these cameras are the keys for head tracking and surface reconstruction. The depth camera is used to draw the digitalized surroundings by measuring the distance and depth. The depth camera also supports the Kinect technology, which uses a combination of camera data and infrared sensing data to capture user body motion and gestures. In addition, Hololens has ambient light sensors to optimize visual quality in different light conditions. Cameras and sensors layout is shown in Figure 3 below.

Figure 3

The most interesting part of Hololens is it's lenses. Hololens has a pair of translucient lenses, so the lenses will not block users from seeing the real world. To visualize virtual objects generate by Hololens, Hololens will emit light from its camera, the light will travel through the lenses, then the light will be reflected and projected into users eyes as particles of light. Images projected are 2D images; however, each lens will project light into different images to create a stereoscopic illusion of 3D.


How Does it work?

The process:
  • Hololens uses sensors and cameras to scan the environment and user actions
  • Hololens constructs a 3D model of the environment using the captured data
  • Hololens calculates head position, and orientation, then determines objects to be rendered
  • Hololens emits light with cameras to the lenses
  • Each lens projects light into different images
  • User sees 3D objects through both lenses, speaker outputs spatial sound, which results in MR experience.

    • The steps are easy to understand, however, some steps are challenging to achieve. As discussed in the previous section, Hololens has many cameras and sensors, therefore, it has to deal with a large amount of input data. The device also needs to track user's head position, orientation, and movement speed altogether to generate different objects views in different angles. Overall, there will be terabytes of data that needs to be processed, and Hololens needs to continuously handle these data in real time in order to provide MR experience to users.

    Hololens requires a large amount of processing power as it is designed to process millions of data items in milliseconds. However, since Hololen is a portable device, it has spatial limitation to hardware to be installed. Hololens has all of its CPU, GPU, RAM, WiFi chip, and bluetooth chip installed on a tiny board. Hardware spec can be viewed in Figure 4 below. Hololens uses a 32-bit intel based CPU with clock speed of 1.04GHz; it is obviously not a super processor, how can Hololens meet the real time requirements?


    Figure 4

    It is noticeable in Figure 4 that Hololens has a special processing unit called the HPU (Holographic processing unit), which is designed for holographic processing as its name indicates. The HPU is reported to process terabytes of Hololens sensor data in real time. As the result, CPU is mainly used for OS tasks and scheduling, understanding the surrounding environment and rendering 3D objects into user's view are all handled by Microsoft's custom made HPU. The HPU has the power to translate Hololens sensor data into much smaller size data to process. Unfortunately, the HPU is also called the "magic black box thingy", details about HPU are not revealed by Microsoft.


    Summary

    In summary, Hololen is a power full device that can bring Mixed Reality experience to users. Its capability of merging different sensors, cameras and other required hardware components into a small piece of head equipment is stunning. Hololens' magic black box HPU is the key for the success of this device; although details are not revealed to the public, Hololens' performance and potentials has prooven to the public that modern technology is capable of processing terabytes of data in real time.
