Introduction

The goal of project 1 consists of two main phases and the respective goals for each phase are to understand how to use and program hardwares and their I/O architecture using the arduino libraries, and to be familiar with UART wireless communication and time scheduling concepts. By the end of project 1, users should be able to use the joystick to control the Roomba, using another joystick to control the pan-and-tilt servos, and start the laser pointer with a push of joystick.

The objective of phase 1 is to learn how to connect all hardware components except bluetooth to a single station. An LCD display will show the input of light sensor value as well as joystick value. The learning objective is to learn how all the devices connect to ATMega 2560 board and get familiar with Arduino library.

The phase 2 of project will split the system into a base station and a remote station. The objective of phase 2 is to learn how to establish a connection between Bluetooth, understand how time-triggered scheduler works. The Roomba will perform actions that base on incoming commands.
